# API de Red Social

## Instalación
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/sbtn63/api-basic-social.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```

## Uso
Iniciar el servidor:
```bash
npm start
```
Hacer solicitudes a la API utilizando herramientas como Postman o curl.

## Características
- Autenticación de usuarios.
- Creación y gestión de publicaciones.
- Interacción entre usuarios.

## Detalles Técnicos
- Lenguaje de programación: JavaScript (Node.js).
- Framework: Express.
- Base de datos: MongoDB.
